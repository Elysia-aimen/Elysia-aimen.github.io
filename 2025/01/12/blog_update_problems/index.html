在更新blog时遇到的各种问题与解决方案

1，使用哔哩哔哩插件下载不到本地

node版本过低导致的，卸载node，下载npm，在npm上下载node并可以下载多个版本选择一种使用

 

2，下载npm与合适node版本后无法使用hexo命令

环境配置的问题，需要重新配置环境变量

3，重新配置环境变量之后依旧无法使用环境变量

需要在新环境下下载执行

npm install webpack -g

安装hexo cli

验证是否对命令响应
